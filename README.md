# A Valentine's Advent Calendar

This is a simple, interactive advent calendar created as a Valentine's Day gift. Each day from February 7th to February 14th, a new door unlocks to reveal a small game or message.

## Features

*   **Interactive Envelope:** The experience begins with a clickable envelope that opens to reveal the advent calendar.
*   **Daily Unlocks:** Doors for each day unlock automatically, building anticipation.
*   **Mini-Games:** Each day reveals a unique, small game or interactive experience.
*   **Festive Design:** The project features a cute, Valentine's-themed design with animations and confetti effects.

## How to Use

1.  Clone this repository.
2.  Open the `index.html` file in your web browser.
3.  Click the envelope to get started.
4.  Click on the unlocked doors to see the surprise for each day.

*Note: The doors are unlocked based on the current date. To see all the content, you can either change your system's date or modify the `script.js` file.*

## Technologies Used

*   HTML5
*   CSS3 (with animations)
*   JavaScript
*   [canvas-confetti](https://github.com/catdad/canvas-confetti) for the fun confetti effects.
